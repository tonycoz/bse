Catalog Editor

Parent

The parent section this article will be placed under.

Catalogs can be moved to other shop sections by selecting a different parent shop section from this menu.

Catalog title

The title of the catalog. (The ‘title’ is indexed by the built-in search engine).

Template

The HTML template used to layout the display of this catalog.

Body

The content of your catalog. There are a number of tags you can use to markup your article text. (The ‘body’ is indexed by the built-in search engine).

Release date

The date that your catalog is released to the world. Your article will only appear in menus after this date and before the expiry date. If you don't enter a release date it will default to the current date.

Note: Unreleased catalogs can be accessed via a direct URL to the catalog.

Expiry date

The date from which you don't want the catalog to appear on your site.

Note: Expired catalogs can be found through the built-in search engine.

Summary length

The length of the summary displayed for sub catalogs by the <:summary:> tag.

Display threshold

This is used by the <:if UnderThreshold :> conditional tag. If the number of sub-catalogs is less than this value the condition is true.

Typically your templates will embed sub-catalogs if there are fewer than this number of sub-catalogs, otherwise a menu of sub-catalogs will be displayed.

Keywords

This is a comma-separated list of extra keywords used by the built-in search engine.

Keywords only need be explicitly indicated here if this catalog should be indexed under words not already used in the ‘title’ or ‘body’ of this catalog.

List article

Choose how the catalog is listed in the sidebar menus, or in lists of sub-catalogs in the parent.

Thumbnail image

An image that can be used with the <:thumbnail:> tag to display a small image for the catalog in menus.

To upload a ‘thumbnail’ image, either enter the name of an image on your computer, or use the Browse button to select an image file (Usually with a .GIF, .JPG, .PNG file extension) then remember to update the catalog or it wont be uploaded and saved.

Images

Images attached to the product. This isn't available until the product has been created.

You can upload and modify the images by clicking the "Manage Images" link.
