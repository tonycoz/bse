Send Subscription

Title/Subject

The title that will be used when you send out the newsletter. This is also used as the subject of the email.

Body

The body text for the newsletter. You can use the normal body text markup. The same text is used for the HTML and text versions of the message. The markup only has an effect for the HTML version, all markup is removed for the text version of the email.

HTML Template

This is used to layout the HTML part of your message.

Text Template

This is used to layout the text part of your message.

Archiving

The options below this heading control whether and how the message will be archived as an article. You can use this to keep an online archive of all of the messages you've sent to your subscribers.

Archive

If this is checked then a copy of the message will be saved as an article.

Keywords

The keywords that will be used for the archived article.

Article Template

The name of the article template that will be used for the archived message.

Archive Parent

The parent article for the archived message.

Testing

The values you enter here are used only for sending test messages. By sending a test message you can check how the message will look in certain email clients, and track down template or processing problems more easily.

Test Name

A name for the recipient.

Test Email

The address the test will be sent to.

Test Text Only

If this is set it's as if the subscriber had checked "Text Only Emails" in their user profile.

Send Subscription

Sends the newsletter to the subscribers.

Send Test

Sends a test copy of the the newsletter to the Test Name and Test Email you specify. The message is not archived.

HTML Preview

Displays a preview of the HTML part of the newsletter in the bottom frame.

Text Preview

Displays a preview of the text part of the newsletter in the bottom frame.
