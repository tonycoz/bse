Subscriptions Manager

Name

The name of the subscription as it appears in the list given in the user options, and in the subscriptions list.

Title

The default title that will be used when you send out a newsletter using this subscription. You can modify this when you send the newsletter.

Description

This text can be used in your templates to describe the file.

Frequency

The frequency you will be sending the newsletter. This should be a simple description like Weekly, or Every Monday and Thursday.

HTML Template

This is used to layout the HTML part of your message.

Text Template

This is used to layout the text part of your message.

Archiving

The options below this heading control whether and how the message will be archived as an article. You can use this to keep an online archive of all of the messages you've sent to your subscribers.

Archive

If this is checked then a copy of the message will be saved as an article.

Keywords

The keywords that will be used for the archived article, if you choose to archive messages in this subscription.

Article Template

The name of the article template that will be used for the archived message.

Archive Parent

The parent article for the archived message.
