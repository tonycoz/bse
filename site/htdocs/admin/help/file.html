File Wizard

File to add

Either enter the name of a file on your machine, or use the Browse button to select a file.

Please note that if you select a large file, it may take several minutes to upload to the web server unless you have a large link. eg. on a 56K link a 1 megabyte file will take 4 minutes to upload, assuming you have a perfect ISP.

Description

This text can be used in your templates to describe the file.

Content Type

This is the format that the users browser will open the file as. BSE understands common types such as PDF, text and HTML, and more can be configured.

If BSE cannot guess the type of the file it will be set to application/octet-stream, ie. a binary file

BSE guesses the file type based on its extension, if BSE's built-in list of extensions is not matching your file, you can add extra entries to the [content types] section of the configuration file.

Treat as download

If this checkbox is checked then BSE will attempt to make the users browser download the file rather than display it.

This can be useful for some types such as PDF files, since displaying PDFs can destabilize some browsers.
