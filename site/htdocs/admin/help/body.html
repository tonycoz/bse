Body Text

BSE uses the same tags for the body text in articles, products and subscriptions.

a[url,text]
link[url]
link[url|text]
poplink[url]
poplink[url|text]
Creates a link to the given url using text as the text of the link. poplink[] will open the link in a new browser window. If no text is provided the url is used as the text of the link.
eg. "link[http://www.develop-help.com/bse/|BSE]" is formatted as "BSE"
b[text]
Formats text in bold. eg. "b[hello]" is formatted as "hello".
i[text]
Formats text in italics. eg. "i[hello]" is formatted as "hello".
image[index]
image[index|alignment]
image[index|alignment|url]
image[index|alignment|url|style]
Inserts the image number index associated with the article. index counts from the top starting with 1. If no alignment is provided then the image is shown inline. If no url is provided then the images URL is used if there is one. If either url is present the image becomes a link, though no border is shown around the image.
The style parameter can be either a CSS class name, inserted as a class="style" parameter, the parameters to the CSS style padding, inserted as a style="padding: style or as CSS style commands inserted as a style="style" parameter.
If you use the image[] tag then the normal automatic image insertion is not done.
image[name]
image[name|alignment]
image[name|alignment|url]
image[name|alignment|url|style]
Same as above, but refers to the images by name. This does not prevent the normal automatic image placement.
gimage[name]
gimage[name|alignment]
gimage[name|alignment|url]
gimage[name|alignment|url|style]
Inserts the given global image where the image is identified by name.
** line1
** line2
Formats the text with bullets. The above would be formatted as:
  • line1
  • line2
## line1
## line2
Formats the text as a numbered list. The above would be formatted as:
  1. line1
  2. line2
%% line1
%% line2
Formats the text as an alphabetical list. The above would be formatted as:
  1. line1
  2. line2
fontcolor[size|color|text]
Formats text with the given size and color. eg. "fontcolor[+1|#FF4040|hello]" is formatted as "hello".
indent[text]
Indents text by putting an empty unordered list around it. eg. "indent[hello]" is formatted as:
    hello
center[text]
Centers text on the line. eg. "center[hello]" is formatted as:
hello
align[alignment|text]
Aligns the text based on alignment which can be any of left, center or right. eg. "align[center|hello]" is formatted as:
hello
font[size|text]
Formats text at the given size, which can be a number from 1 through 7 for absolute sizes, or + or - followed by a number for a relative font size.
eg. "font[+1|hello]" is formatted at "hello".
hr[width]
hr[width|height]
Inserts a horizontal line into the page, width wide and height high.
eg. hr[50,4] produces:
anchor[name]
Produces an invisible anchor for a link with the given name.
table[width|bgcolor|padding|fontsize|fontface
rowopts|cell|cell...
rowopts|cell|cell...
...
]
table[name=value ...
rowopts|cell|cell...
rowopts|cell|cell...
...
]
Inserts a simple table into the page. In the first form each value corresponds to the equivalent table option, except that the fontsize and fontface options are used to format the text in each cell.
The rowopts are <tr> tag attributes, and can often be left empty.
eg. "table[width="50%" border=1
|a|b
|c|d
]" produces
ab
cd
hrcolor[width|height|color]
Produces a box of the given width and height and of the given color. eg. hrcolor[20|20|#00FF00] produces:
embed[article]
embed[article,template]
embed[article,template,maxdepth]
Inserts the text from the given article, which can be an article number or the name of a tag that refers to an article.. Uses the specified template, or the article's template if none is supplied.
html[text]
The text is used exactly as is, as HTML. eg. "html[<b>hello</b>]" is formatted as "hello"
This can be used for inserting much larger pieces of HTML, such as tables.
doclink[articleid]
doclink[articleid|title]
popdoclink[articleid]
popdoclink[articleid|title]
Creates a link to the article specified by articleid which must be either the number id of the article, or a symbol defined in the [articles] section of the configuration file, for example 3 or shop.
If you do not supply a title the article title is used instead.
popdoclink[...] does the same as doclink[...] except that the link is opened in a new window.
class[classname|text]
Marks text to be formatted using CSS class class.
style[style code|text]
Renders text as if it were in a CSS class that included style code in its definition.
popimage[inlineimage|popupimage]
popimage[inlineimage|popupimage|align]
popimage[inlineimage|popupimage|align|style]
Inserts the given inlineimage into the displayed text. Aligned as be align and styled as per style. The align and style behave as with the image[...] tag.
When the user clicks on the inlineimage a popup window is displayed containing popupimage.
If either of the images cannot be found, then nothing is displayed by this tag.
